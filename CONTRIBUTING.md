# Contributing

Thanks for your interest in j5ed-knowledge-graph!

## Getting Started

```bash
git clone https://github.com/ArkTechNWA/j5ed-knowledge-graph.git
cd j5ed-knowledge-graph
npm install
npm run build
npm test
```

## Development

- **Build:** `npm run build`
- **Test:** `npm test` (7 suites, 80 tests)
- **Lint:** `npm run lint`
- **Format:** `npm run format`

## Submitting Changes

1. Fork the repo and create a branch from `main`
2. Make your changes
3. Ensure `npm run build && npm test && npm run lint` all pass
4. Open a pull request against `main`

## Code Style

- TypeScript strict mode
- Prettier for formatting
- ESLint 10 with flat config

## Reporting Issues

Use [GitHub Issues](https://github.com/ArkTechNWA/j5ed-knowledge-graph/issues). For security vulnerabilities, see [SECURITY.md](SECURITY.md).
